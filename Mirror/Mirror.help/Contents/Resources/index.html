Mirror - Help

Mirror is a web-scraper/spider allowing you to backup ('mirror') websites for offline use.
This could prove useful if you want to have a backup of some website or copy a website that may be shut down.

Mirroring To start mirroring a link, enter one in the project window and press enter. The application starts mirroring automatically, reporting on the number of bytes that have been received and written to disk, the transfer rate, ... You can add more links. They are kept by the project and mirrored when deemed appropriate.

Files that are being downloaded (or have been downloaded), errors, and warnings are displayed in the statistics window which can be accessed by clicking View > Show Statistics.
A project can be paused and its files can be browsed through. Its contents can be saved 'as is' (with the structure format chosen in the project options), or as a bundle. The latter behaviour can be altered in the global app preferences, which can be accessed through the application's main menu (Mirror > Preferences...).

Options The app includes many options, including :
  • Spider : alter the ways robots.txt files are treated, parse JAVA files, accept cookies, attempt to parse links even in javascript or other tags, ...
  • Limit : set the maximum mirroring depth (both internal and external), the maximum size of non-HTML or HTML files that are to be mirrored, the maximum size of the mirror, the maximum time spent mirroring, the maximum transfer rate, the maximum connections/second, the maximum number of links, ...
  • Download : set the maximum number of connections, the timeout length, the number of retries (when the app fails to download a file), the minimum transfer rate, ....
  • Identity : use a proxy, set up the default referrer URL, set up HTTP headers, ...
  • Build : create log files, use a cache, alter the structure of the resulting copy, make an index, ... (websites can be downloaded 'as is' or can also be encapsulated in document files)
These options can be set by clicking View > Show Options. This opens a window with project options, where most options are described by their tooltip - you know, that little popup that becomes visible once you hover over something.
It is also possible to save your custom options as a preset. To do so, make sure the options window is active, and use the Project > Options > Save Preset... menu.

More Information You are welcome to request new features or ask questions about the app - contact information is displayed in the application's 'About' window (Mirror > About Mirror).
